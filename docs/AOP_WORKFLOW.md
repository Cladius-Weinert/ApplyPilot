# AOP Research Workflow

Safe flow:
1. Intake study.
2. Read guideline.
3. Extract evidence.
4. Log source.
5. Draft submission.
6. Human review.
7. Assisted highlight.
8. Final approval.
9. Manual submission.

Rules:
- Do not guess.
- Unknown fields: Unknown / Leave blank.
- Every claim needs evidence.
- No automatic final submission.
